\textbf{ Eisenstein\_2-block }

\label{ex:Eisenstein2-block}

The elements $ \left\{2\omega - 1, 2\omega, \omega + 1, -\omega - 1, -2\omega, -2\omega + 1\right\} $ have no representative  modulo $\beta-1$ in the alphabet $\mathcal{A}$.
